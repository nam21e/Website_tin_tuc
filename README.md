# 📰 Website Tin Tức - Back-end

## 📌 Giới thiệu
Đây là phần **Back-end** của hệ thống Website Tin Tức.  
Hệ thống cung cấp các API để quản lý:
- Người dùng (User)
- Bài viết (Article)
- Danh mục (Category)
- Nguồn tin (Source)
- Upload ảnh
- Xác thực và phân quyền

---

## 🚀 Công nghệ sử dụng
- Node.js
- Express.js
- Supabase (Database)
- JWT (Authentication)
- Multer (Upload file)
